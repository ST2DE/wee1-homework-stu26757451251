可以選擇各種不同 HTTP 請求的種類

example. 使用 GET 的方式，發送請求給 Web Service:
            1.打上URL 
            2.Web Service 回傳資料結果
            
可以在 Params 的部分添加參數

SAVE: 可將執行過的請求儲存在 Collection
Status: 回傳的狀態 
time:　時間
在 Body 部分，也可以調整回傳結果的格式 for instance: XML,json,HTML...etc

=> 建議要用幾張圖，來說明怎麼操作
=> 至少要解釋怎麼用到「GET」/「POST」
